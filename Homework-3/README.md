# АСОИУ: домашнее задание №3

Две реализации приложения на C# для варианта 22:

- справочник: рестораны;
- основная таблица: блюда в меню;
- числовой показатель: цена блюда в рублях.

Проекты:

- `ASOIU-3` — консольный интерфейс;
- `ASOIU-3.WinForms` — графический интерфейс Windows Forms для строгого
  соответствия требованиям ТЗ.

Обе реализации используют общие модели, сервисы, Entity Framework Core и
SQLite.

## Возможности

- автоматическое создание SQLite-базы через `EnsureCreated`;
- начальное заполнение: 4 ресторана и 12 блюд;
- CRUD для ресторанов и блюд;
- запрет удаления ресторана со связанными блюдами;
- выбор ресторана из списка при добавлении и изменении блюда;
- проверка, что цена неотрицательна;
- отчёт с полным списком, количеством и средней ценой по ресторанам;
- обработка ошибочного ввода без аварийного завершения.

## Запуск консольной версии

```bash
cd ./ASOIU-3 && dotnet run
```

## Запуск WinForms-версии

WinForms-приложение запускается только на Windows:

```bash
cd C:\ASOIU\Homework-3
dotnet run --project ASOIU-3.WinForms/ASOIU-3.WinForms.csproj
```

В графическом интерфейсе предусмотрены три раздела:

- рестораны — просмотр, добавление, изменение и удаление;
- блюда — просмотр, добавление, изменение и удаление с выбором ресторана;
- отчёт — полный список, количество блюд и средняя цена по ресторанам.

Для открытия обоих проектов в Visual Studio или VS Code используется файл
`ASOIU-3.slnx`.
