## Задачи из лекций

| № | Проект | Краткое описание | Команда запуска |
|---|--------|------------------|-----------------|
| 1 | Task01_BiquadraticSolver | Решение биквадратного уравнения и сортировка корней | `dotnet run --project Task01_BiquadraticSolver` |
| 2 | Task02_ObjectArrayPatterns | Анализ массива `object[]` через `if`, `switch` и `switch expression` | `dotnet run --project Task02_ObjectArrayPatterns` |
| 3 | Task03_ArrayLoops | Операции с массивом через разные циклы | `dotnet run --project Task03_ArrayLoops` |
| 4 | Task04_MatrixSymmetry | Проверка квадратной матрицы на симметричность | `dotnet run --project Task04_MatrixSymmetry` |
| 5 | Task05_NullableDemo | Демонстрация `null`, nullable-значений и операторов `??`, `?.`, `??=` | `dotnet run --project Task05_NullableDemo` |
| 6 | Task06_TxtFileFinder | Поиск `.txt` файлов в каталоге и подкаталогах | `dotnet run --project Task06_TxtFileFinder` |
| 7 | Task07_NotesFileManager | Менеджер заметок в файле `notes.txt` | `dotnet run --project Task07_NotesFileManager` |
| 8 | Task08_StringBuilderList | Формирование нумерованного списка через `StringBuilder` | `dotnet run --project Task08_StringBuilderList` |
| 9 | Task09_StreamPipeline | Работа со стандартными потоками в режимах `generate` и `filter` | `dotnet run --project Task09_StreamPipeline -- generate` |
| 10 | Task10_StudentCollections | Учет студентов на `List`, `Stack`, `Queue`, `Dictionary` | `dotnet run --project Task10_StudentCollections` |
| 11 | Task11_LibraryBasics | Базовая библиотечная система с классом `Book` | `dotnet run --project Task11_LibraryBasics` |
| 12 | Task12_LibraryOOP | ООП-версия библиотечной системы | `dotnet run --project Task12_LibraryOOP` |
| 13 | Task13_LibraryAdvanced | Продвинутая библиотечная система с исключениями, интерфейсами, enum и generic-каталогом | `dotnet run --project Task13_LibraryAdvanced` |
| 14 | Task14_LinqObjectsFinal | Итоговые LINQ to Objects запросы по сотрудникам, отделам и проектам | `dotnet run --project Task14_LinqObjectsFinal` |
